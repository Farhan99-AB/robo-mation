COMMANDS

To open virtual env(always be in the project directory)
zephyrproject\.venv\Scripts\activate.bat 

TO check contents of a file 
type zephyr\samples\hello_world\src\main.c 

to build files
west build -b esp32_devkitc/esp32/procpu -p always --sysbuild zephyr/samples/hello_world

west build -p always -b esp32_devkitc/esp32/procpu --sysbuild apps/chatterbox

west build -p always -b esp32_devkitc/esp32/procpu --sysbuild apps/tda

west build -p always -b esp32_devkitc/esp32/procpu --sysbuild apps/edf 

west build -p always -b esp32_devkitc/esp32/procpu --sysbuild apps/pcp

USE THIS COMMAND WHICH SPECIFIES WHICH OVERLAY FILE TO USE 
west build -b esp32_devkitc/esp32/procpu -p always --sysbuild apps/chatterbox -- -DDTC_OVERLAY_FILE=esp_wrover_kit_procpu.overlay

DOUBT


to flash 
west flash --esp-device COM11

to open serial monitor
west espressif monitor -p COM11
